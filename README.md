# E-Kunduluk Training

Тренажер для учителей — эмуляция интерфейса электронного журнала Э-Күндөлүк.

**Открыть:** https://ninenko.github.io/e-kunduluk-training/

## Что отрабатывается
- Выбор столбца с датой урока (клик по шапке или по любой ячейке).
- Кнопка «Выставление оценок» появляется только при выбранном столбце.
- Окно выставления: выбрать оценку → нажать «+» напротив ученика.
- Удаление: выбрать 🗑 → нажать на оценку ученика.
- Все посторонние кнопки отвечают «Это не нужно для выставления оценок».

Имена учеников и данные урока — вымышленные.
